## Parameters

### Using `AllowedValues`
```yaml
Parameters:
  InstanceType:
    Description: Choose EC2 instance type
    Type: String
    Default: t2.micro
    AllowedValues:
    - t2.micro
    - t2.medium
    - t2.large
```

```yaml
Resources:
  MyInstance:
    Type: AWS::EC2::Instance
    Properties:
      InstanceType: !Ref InstanceType
      InstanceId: ami-0c02fb55956c7d316
```

### Using `NoEcho`
```yaml
Parameters:
  DBPassword:
    Description: This is DB password
    Type: String
    NoEcho: true
```
```yaml
Resources:
  MyDBInstance:
    Type: AWS::RDS:DBInstance
    Properties:
      DBInstanceClass: db.t2.micro
      AllowedStorage: 20
      Engine: mysql
      MasterUsername: admin
      MasterUserPassword: !Ref DBPassword
      DBInstanceIdentifier: mydbinstance
```
