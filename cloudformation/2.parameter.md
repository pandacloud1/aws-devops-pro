## Parameter

### Using `AllowedValues`
```yaml
Parameters:
  InstanceType:
    Description: Choose EC2 instance type
    Type: String
    Default: t2.micro
    AllowedValues:
    - t2.micro
    - t2.medium
    - t2.large
```

```yaml
Resources:
  MyInstance:
    Type: AWS::EC2::Instance
    Properties:
      InstanceType: !Ref InstanceType
      InstanceId: ami-0c02fb55956c7d316
```
